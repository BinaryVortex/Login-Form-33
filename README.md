# Login Form

A clean, responsive login form built using HTML, CSS and JavaScript. This repository contains a simple, modern-looking login UI that can be used as a starting point for websites and small projects.

![Login Form Screenshot](./Screenshot%202024-07-02%20102520.png)

## Demo
Open index.html in your browser to see the login form in action.

## Features
- Simple, modern design
- Responsive layout
- Built with plain HTML, CSS and a small amount of JavaScript
- Background image included (bg.jpg)

## Files
- `index.html` — Main HTML file
- `style.css` — Styles for the login form
- `bg.jpg` — Background image used by the page
- `Screenshot 2024-07-02 102520.png` — Screenshot shown above

## Usage
1. Clone the repository:

   git clone https://github.com/BinaryVortex/Login-Form-33.git

2. Open `index.html` in your preferred browser.

No build tools or package managers are required.

## Customize
- Edit `style.css` to adjust colors, spacing, and typography.
- Replace `bg.jpg` to change the background.
- Update `index.html` to integrate the form into a larger project or to hook it up to a backend authentication endpoint.
